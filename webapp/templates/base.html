About this tool

What is Cell Painting?

Cell Painting is a microscopy assay that stains cells with multiple fluorescent dyes marking different structures — nucleus, endoplasmic reticulum, RNA, the cytoskeleton/Golgi/membrane, and mitochondria. Imaging all of these together captures a detailed snapshot of a cell's morphology after treatment with a compound.

What did the model learn?

A self-supervised vision model (DINO) was trained on these images without any mechanism-of-action labels, learning to compress each image into a 384-number embedding. For details on how the map itself is built and how to read distance and similarity, see "ⓘ About this visualization" in the left sidebar.

Technical details

Embeddings were created from JUMP Cell Painting wells and postprocessed with MAD and sphering. Similarity is determined via a simple cosine similarity search.

Acknowledgements

This project builds upon several open datasets, research efforts, and visualization tools. The microscopy data were provided by the JUMP Cell Painting Consortium (JUMP-CP), an open resource for large-scale exploration of cellular phenotypes using high-content imaging.

The self-supervised representation learning approach was inspired by Kim et al. (2025), "Self-supervised learning of morphological representations from large-scale Cell Painting data", which demonstrated the potential of vision models to learn biologically meaningful representations from Cell Painting images without relying on mechanism-of-action labels.

The interactive embedding visualization was inspired by the TensorFlow Embedding Projector , an open-source tool for exploring high-dimensional embeddings through interactive dimensionality reduction and similarity-based visualization. Its design principles informed the interface used to explore learned cellular representations in this tool.
